* Active Directory Setup
Como instalei a versão core, vou fazer tudo via PowerShell, após já ter definido o nome para a máquina, e realizar as outras configs iniciais, no powershell ->
#+begin_src powershell
Install-WindowsFeature AD-Domain-Services -IncludeManagementTools
#+end_src

Setup do Domain Controller (DC) -->
#+begin_src powershell
Install-ADDSForest -DomainName "lab.local"
#+end_src

Criando uma nova OU (Organizational Unit) -->
#+begin_src powershell
New-ADOrganizationalUnit -Name "Matriz" -Path "DC=dc01,DC=lab"
#+end_src

Criando um novo user nesta OU -->
#+begin_src powershell
New-ADUser -Name "biel" -SamAccountName "biel.lab" -Path "OU=Matriz,DC=dc01,DC=lab"
#+end_src
** ssh setup
Instalar o OPenSSH -->
#+begin_src powershell
Add-WindowsCapability -Online -Name OpenSSH.Server
#+end_src

Iniciar o serviço -->
#+begin_src powershell
Start-Service sshd

Set-Service -Name sshd -StartupType 'Automatic'
#+end_src
